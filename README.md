# Apex Map Rotation

Apex Legendsのマップローテーションスケジュールを確認できるWebツールです。
現在のマップや残り時間、今後のスケジュールをブラウザ上で簡単に確認できます。

## 🌟 特徴 (Features)

* **リアルタイム表示**: 現在プレイ可能なマップ、残り時間、進行状況（プログレスバー）をひと目で確認できます。
* **スケジュール一覧**: 今後のマップローテーションをタイムライン形式で表示します。
* **日付選択・カレンダー機能**: タブやカレンダーから、特定の日付のスケジュールを確認可能です。
* **マップ絞り込み（フィルター）**: 「ワールズエッジのみ」など、特定のマップの予定だけを抽出して表示できます。
* **レスポンシブ対応**: PCだけでなく、スマートフォンからも快適に閲覧できるデザインです。

## 💻 使用技術 (Tech Stack)

* HTML5
* CSS3 (Custom Properties, Grid, Flexbox)
* Vanilla JavaScript
* ※外部ライブラリやフレームワーク（React, Vueなど）は使用していません。
