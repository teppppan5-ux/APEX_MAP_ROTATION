# Apex Map Rotation

Apex Legends のマップローテーションスケジュールを確認できるWebアプリケーションです。

## 特徴
- 現在プレイ可能なマップと残り時間をリアルタイムに表示
- 日付別のマップローテーションスケジュールの確認
- 特定のマップでの絞り込み機能

## 公開方法 (GitHub Pages)

1. このフォルダ（`apex-map-rotation`）を GitHub リポジトリとしてプッシュします。
2. GitHub リポジトリの **Settings** > **Pages** を開きます。
3. **Source** を `Deploy from a branch` に設定し、`main`（または `master`）ブランチの `/ (root)` を選択して **Save** をクリックします。
4. 数分待つと、GitHub Pages の URL (`https://<あなたのユーザー名>.github.io/apex-map-rotation/`) でアクセスできるようになります。

## 構成
- `index.html`: アプリケーションのメインファイル (HTML, CSS, JavaScript が含まれています)
